Tentang Saya

Saya adalah individu yang tertarik pada teknologi, website, dan pengembangan sistem. Website ini dibuat sebagai profil profesional dan tempat berbagi perjalanan belajar saya. Saya percaya bahwa pembelajaran mandiri adalah kunci untuk berkembang di era digital ini.

Riwayat Pendidikan

Saya menyelesaikan pendidikan Sarjana Teknik Informatika di Universitas Indonesia pada tahun 2018-2022. Sebelumnya, saya menempuh pendidikan menengah atas di SMA Negeri 1 Jakarta dari tahun 2015-2018. Pendidikan ini memberikan dasar kuat dalam bidang teknologi dan pengembangan perangkat lunak.

Keahlian & Minat

Saya memiliki keahlian dalam pengembangan web menggunakan HTML, CSS, dan JavaScript untuk membuat aplikasi interaktif. Selain itu, saya tertarik pada Internet of Things (IoT) dan telah mengembangkan beberapa proyek menggunakan sensor dan platform seperti Arduino. Saya juga memiliki kemampuan dalam pembelajaran mandiri serta analisis sistem, dan saya senang mendesain antarmuka pengguna yang sederhana namun efektif.

Pengalaman

Saya telah mengembangkan berbagai proyek pribadi, termasuk alat deteksi kebisingan menggunakan sensor dan platform Arduino untuk aplikasi IoT. Selain itu, saya memiliki pengalaman dalam membuat website statis untuk klien kecil dengan fokus pada desain responsif dan pengalaman pengguna yang optimal. Pengalaman ini telah membantu saya memahami siklus pengembangan perangkat lunak dari konsep hingga implementasi.

Proyek

Saya telah mengembangkan beberapa proyek menarik, termasuk website portofolio ini yang dibangun dengan HTML, CSS, dan JavaScript untuk interaktivitas. Selain itu, saya memiliki proyek aplikasi IoT untuk memantau suhu dan kelembaban ruangan secara real-time menggunakan ESP32. Proyek-proyek ini menunjukkan kemampuan saya dalam menggabungkan teknologi web dan IoT untuk menciptakan solusi praktis.
